## Simple Text Editor Using Python Tkinter

## Overview
This project is a basic desktop text editor built using Python and the Tkinter GUI library.  
It allows users to create open edit and save text files through a clean menu driven interface.

The project demonstrates fundamental GUI concepts and file handling in Python.

## Features
Create a new file by clearing the text area  
Open existing text files from the system  
Edit text with word wrapping enabled  
Save edited content to a file  
Menu based navigation  
Dark theme editor interface  

## Technologies Used
Python  
Tkinter  
Tkinter filedialog  
Tkinter messagebox  

## Application Components

### File Menu Functions
New clears the editor for a fresh file  
Open loads an existing file into the editor  
Save writes current text to a file  
Exit closes the application  

### User Interface
Main window created using Tk  
Menu bar for file actions  
Text widget with  
Word wrap enabled  
Large readable font  
Dark background and light text  
Visible cursor on dark theme  

## How the Application Works
The Tkinter main window initializes the editor  
Menu options trigger callback functions  
The Text widget manages user input  
File dialogs handle open and save actions  

## How to Run
Make sure Python is installed  
Save the file as main.py  
Run the application using  

python main.py
